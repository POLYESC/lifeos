# 🏠 우리집 라이프 OS

가족 생활 통합 관리 도구. 모든 데이터는 본인 브라우저에만 저장되며 외부로 전송되지 않습니다.

## 📦 모듈

- 💰 **가계부** - 가계부 + 자산 관리
- ✈️ **여행일지** - 여행 계획 + 추억
- 👶 **자녀 기록** - 임신/육아 + 마일스톤 + 사진
- 🏠 **가전 / 스마트홈** - 가전 통합 관리
- 🥗 **건강 / 식단** - 음식 사진 칼로리 분석
- 🔧 **집안 관리** - 정기 점검 + 수리 이력
- 🤖 **AI 비서** - 모든 모듈 데이터를 활용한 통합 비서

## 🔒 데이터 저장

모든 데이터는 **사용자의 브라우저 localStorage**에만 저장됩니다.
- 외부 서버 전송 없음
- 본인 기기에만 보관
- 코드는 공개되지만 데이터는 안전

## 🛠️ 사용 기술

- 정적 HTML/CSS/JavaScript (백엔드 없음)
- LocalStorage 기반
- Open-Meteo (날씨, 무료)
- Google Gemini API (AI 비서, 사용자 직접 입력)
- Kakao Map API (맛집 검색, 사용자 직접 입력)
