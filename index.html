☁️ 가족 데이터 동기화 설정

Firebase 무료 클라우드를 사용해 가족과 실시간으로 데이터를 공유해요.
한 명이 처음 설정 후, 가족 코드를 다른 멤버에게 알려주면 같은 데이터를 공유할 수 있어요.
console.firebase.google.com → 프로젝트 설정 → 내 앱 → 웹 앱의 firebaseConfig 통째로 복사
🔑 이 코드를 가족 멤버에게도 알려주세요. 같은 코드를 입력한 사람들끼리 데이터를 공유합니다.
⚠️ 보안상 추측 어려운 긴 문자열로 만드세요 (예: kim-2026-AB7xK9)
Life OS · 모든 데이터는 이 기기 브라우저에 저장됩니다
